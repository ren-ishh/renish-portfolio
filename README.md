# Renish R — Portfolio

Personal portfolio website built with Next.js, Framer Motion, and Tailwind CSS.

## Tech Stack
- **Framework:** Next.js 15 + React 19
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion v12
- **Contact:** EmailJS
- **Deployment:** Vercel

## Getting Started
\```bash
npm install
npm run dev
\```

## Environment Variables
Create `.env.local`:
\```
NEXT_PUBLIC_EMAILJS_SERVICE_ID=
NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=
NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=
\```
